# Emitter and Reciever boards

Project contains 4 boards:
* Emitter-1
* Emitter-2
* Reciever-1
* Reciever-2

Reciever boards connects each other with 4 wires. Emitters connects any reciever or each other with 2 wires (G, +5V).
Reciever-2 board has external interface which can be connected to the controller.
